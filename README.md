# Rulate Crawler TUI

Текстовый интерфейс (TUI) для скачивания новелл с сайта `tl.rulate.ru` и преобразования их в формат EPUB.

## 🚀 Возможности

- 📖 Получение списка глав и метаданных с сайта
- ⚡ Многопоточная загрузка глав и изображений
- 💾 Кэширование глав в JSON или SQLite (на выбор)
- 🖼️ Автоматическое скачивание и вставка изображений в EPUB
- 🎨 Сохранение форматирования текста (жирный, курсив, таблицы)
- 📝 Очистка синопсиса от рекламы и мусорных блоков
- 🔧 Режим отладки с подробными логами
- 💾 Автоматическое сохранение лога (опционально)
- 📱 Оптимизировано для работы в Termux на Android

## 📦 Установка

### Termux (Android)

```bash
pkg update && pkg upgrade
pkg install python git nodejs
pip install -r requirements.txt
git clone https://github.com/ваш_репозиторий/lncrawl-tui
cd lncrawl-tui
python tui_app.py
```

На других платформах

```bash
pip install -r requirements.txt
python tui_app.py
```

🧭 Использование

1. Авторизация – при первом запуске введите логин и пароль от tl.rulate.ru.
2. Добавление новеллы – вставьте URL книги, выберите раздел, подтвердите пути.
3. Выбор глав – отметьте нужные главы или укажите диапазон (например, 1-50, 100).
4. Загрузка – дождитесь окончания; готовый EPUB появится в указанной папке.
5. Настройки – измените пароль, включите режим отладки, выберите тип кэша (JSON/SQLite).

⚙️ Конфигурация

· Логин/пароль – хранятся в ~/.lncrawl.auth.
· Настройки – в ~/.lncrawl_settings.json.
· База библиотеки – ~/storage/shared/lncrawl/Novelsbase/Novels.db.
· Готовые EPUB – по умолчанию в /storage/emulated/0/Novels/ (или папка, указанная при добавлении).

🐛 Отладка

Включите режим отладки в настройках → на экране загрузки появится кнопка «💾 Сохранить лог», а также подробные сообщения о каждом действии. Сохранённый лог поможет при обращении за помощью.
